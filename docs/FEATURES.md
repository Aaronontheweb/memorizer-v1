# Memorizer V2 - Feature Gallery

A visual tour of Memorizer V2's features, shown in both light and dark themes.

---

## Memory Management

### Memory List

Browse all memories with search, filtering, and pagination. Memory cards show title, type, tags, confidence, and timestamps.

| Dark Mode | Light Mode |
|-----------|------------|
| ![Memory List - Dark](images/memory-list-dark.png) | ![Memory List - Light](images/memory-list-light.png) |

### Memory View

View individual memories with full content rendering (including Mermaid diagrams and syntax-highlighted code), relationships to other memories, version history, and similar memories found via semantic search.

| Dark Mode | Light Mode |
|-----------|------------|
| ![Memory View - Dark](images/memory-view-dark.png) | ![Memory View - Light](images/memory-view-light.png) |

### Create Memory

Create new memories with a rich form including type selection, workspace/project assignment, tag input, and confidence scoring.

| Dark Mode | Light Mode |
|-----------|------------|
| ![Create Memory - Dark](images/create-memory-dark.png) | ![Create Memory - Light](images/create-memory-light.png) |
